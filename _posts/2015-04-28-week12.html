Technical Guide

Set up a basic InDesign file with 10 pages

Open InDesign ==> File ==> New ==> Document

Set Number of Pages to 10, and uncheck Facing Pages

Choose Intent ==> Web

To place external graphic file

File==>Place==>drop==>Hold Shift and resize outline box==>Click Fit Content Proportionally

To add and/or delete pages

Window==>Pages==>Control+click any spread inside A-Master, choose insert or delete.

Work with Master File

Go pages==>A-Master File and double click the icon to the right to start editing master file.

To avoid Master File format being applied to certain pages, go to Pages==>Right Click the Spread Icon==>Apply Masters to pages==>None.

To edit a Master File format on a specific spread, hold CMD+SHIFT+Click the field's dialoug box.

To draw and color a rectangle

select rectange tool==>draw==>adjust Fill and Stroke Color

File Export

File==>Export==>Choose Format ADOBE PDF Interactive==>Change JPEG quality to high.
